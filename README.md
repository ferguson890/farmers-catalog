# Каталог производителей — Ассоциация рестораторов Нижегородской области

Каталог фермеров и производителей региона: карточки с фильтрацией и поиском + отдельная страница каждого производителя. Данные хранятся в одном JSON-файле, интерфейс генерируется из него. Цель — лёгкое наполнение каталога без участия разработчика и публикация на [Tilda](https://nireal.ru).

> Визуальный референс — сайт Ассоциации **nireal.ru**. Функциональный референс — исходный HTML-прототип (см. `design-reference/`).

---

## Возможности

- **Каталог** (`index.html`) — сетка карточек, фильтр по категориям, живой поиск (название / район / продукт / производитель).
- **Страница производителя** (`farmer.html?id=<id>`) — обложка, описание, история основателя, продукция, преимущества, контакты, фотогалерея.
- **Единый источник данных** — `data/farmers.json`. Чтобы добавить производителя — добавить один объект в массив. Кода не трогаем.
- **Адаптивность** — desktop / tablet / mobile.
- **Без зависимостей** — чистый HTML/CSS/JS, разворачивается на любой статике (GitHub Pages, Tilda, Netlify).

## Быстрый старт (локально)

`fetch()` не работает с `file://`, поэтому нужен локальный сервер:

```bash
# Python (есть на машине)
cd farmers-catalog
python -m http.server 8000
# открыть http://localhost:8000
```

```bash
# или Node
npx serve .
```

## Как добавить нового производителя

1. Подготовьте фото: положите в `assets/images/<id>/cover.jpg` (и при необходимости `1.jpg`, `2.jpg` для галереи). См. `docs/PERFORMANCE.md` про оптимизацию.
2. Откройте `data/farmers.json` и добавьте объект в массив `farmers` (структуру полей см. в `docs/DATA-MODEL.md`).
3. Убедитесь, что `category` совпадает со `slug` из списка `categories`.
4. Проверьте JSON на валидность: `python -c "import json;json.load(open('data/farmers.json',encoding='utf-8'))"`.
5. Закоммитьте — каталог и страница обновятся автоматически.

## Текущие данные

В каталоге **6 производителей** (из реестра Ассоциации) в **4 категориях**: зелень и салаты, молочная продукция, кондитерские изделия, мясная продукция.

⚠ Часть данных требует уточнения у производителей (год основания Gastro-Ferma, фото для ряда карточек, регион Smart Farm — Владимирская обл.). Подробности — в поле `notes` каждого фермера в `farmers.json`.
