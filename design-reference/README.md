# Design Reference

Референсы, на которые опирается дизайн каталога.

## 1. Функциональный референс — HTML-прототип (MVP)
Исходный одностраничный прототип каталога (карточки + фильтры + модалка). Из него взяты:
- структура карточки и сетки;
- логика фильтров по категориям и живого поиска;
- паттерн «карточка → подробнее»;
- адаптивные брейкпоинты (1000px / 680px).

> Положите файл прототипа сюда как `prototype.html` для истории.

## 2. Визуальный референс — nireal.ru (сайт Ассоциации)
Стилистика, которую воспроизводим:
- **Настроение:** чистый минимализм, профессионально, много воздуха.
- **Фон:** преимущественно белый + тёплый кремовый.
- **Акцент:** природный зелёный (агро/натуральность), сдержанная терракота.
- **Логотип:** минималистичный круглый знак слева вверху.
- **Иконография:** монохромные линейные иконки.
- **Структура:** фиксированная навигация → герой с миссией → блоки с иконками → галереи членов/партнёров → новости → формы.
- **Навигация (якоря):** О нас · Новости · Помощь ресторатору · Членство · Партнёры.

## Зафиксированная палитра (см. `src/css/styles.css → :root`)
| Переменная | HEX | Назначение |
|---|---|---|
| `--cream` | `#F9F6F0` | основной фон |
| `--white` | `#FFFFFF` | карточки, шапка |
| `--green` | `#3D6B4F` | основной акцент |
| `--greenlt` | `#EFF4EC` | подложка тегов |
| `--terra` | `#B5602A` | вторичный акцент / предупреждения |
| `--ink` | `#1F1A14` | основной текст |
| `--ink2/--ink3` | `#6B5E4E` / `#9E9082` | вторичный/третичный текст |
| `--line/--line2` | `#E2D9CC` / `#CBBFA8` | границы |

**Типографика:** заголовки — Playfair Display (serif), текст — Lato (sans-serif).
