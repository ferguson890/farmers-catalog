# assets/images

Фотографии производителей.

## Соглашение об именовании
```
assets/images/<id>/cover.jpg   # обложка карточки и страницы (обязательно)
assets/images/<id>/1.jpg       # фото для галереи (опционально)
assets/images/<id>/2.jpg
```
где `<id>` — это `id` производителя из `data/farmers.json` (например `smart-farm`, `brynza`).

## Требования к изображениям
- **Формат:** WebP (предпочтительно) или оптимизированный JPG. Для широкой совместимости можно оставить JPG.
- **Обложка:** ~1200×800 px, обрезка под соотношение карточки (object-fit: cover).
- **Вес:** обложка ≤ 200 КБ, фото галереи ≤ 150 КБ.
- **Без текста на фото** (текст добавляется интерфейсом).

## Текущий статус
Фото пока **не загружены** — в реестре они даны ссылками на Яндекс.Диск (поле `photosSource` в JSON). Пока файла нет, карточка/страница автоматически показывают emoji-fallback (`icon`). Это штатное поведение, ошибки не будет.

См. `docs/PERFORMANCE.md` про оптимизацию изображений.
