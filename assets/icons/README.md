# assets/icons

Иконки интерфейса. Сейчас все иконки — инлайновые SVG прямо в JS (`catalog.js`, `farmer.js`): булавка, календарь, телефон, сайт, почта, чат, карта. Это убирает лишние сетевые запросы.

Если иконок станет много или их захотят переиспользовать — выносите сюда отдельными `.svg` либо собирайте SVG-спрайт `sprite.svg` и подключайте через `<use>`.

Категорийные «иконки» — это emoji в `data/farmers.json` (поля `icon` у категорий и фермеров): 🌿 🧀 🍫 🍪 🥩. Их менять — только в JSON.
