# Модель данных

Файл `data/farmers.json` состоит из трёх частей: `meta`, `categories[]`, `farmers[]`.

## meta
Служебная информация о каталоге.
```json
{ "title": "...", "region": "Нижегородская область", "updated": "2026-06-17", "source": "...", "note": "..." }
```

## categories[]
Единый список категорий. Первой всегда идёт `all` (вкладка «Все»).
| Поле | Тип | Описание |
|---|---|---|
| `slug` | string | Машинный идентификатор. На него ссылается `category` фермера. |
| `label` | string | Полное название (для бейджей/страницы). |
| `icon` | string | Emoji-иконка категории. |
| `nav` | string | Короткая подпись для вкладки фильтра. |

## farmers[] — структура карточки И страницы
Один объект описывает и карточку в каталоге, и полную страницу. Поля, помеченные **(карточка)**, используются в сетке; **(страница)** — на детальной странице; остальные — служебные.

| Поле | Тип | Где | Описание |
|---|---|---|---|
| `id` | string | оба | **Уникальный** slug. Используется в URL `farmer.html?id=<id>` и в путях к фото. Латиница, через дефис. |
| `num` | number | служ. | Порядковый номер из реестра. |
| `icon` | string | оба | Emoji-fallback, если нет фото. |
| `name` | string | оба | Название хозяйства/компании. |
| `owner` | string | оба | Руководитель / основатель. |
| `category` | string | оба | Slug категории (обязан существовать в `categories`). |
| `categoryLabel` | string | оба | Человекочитаемая категория. |
| `district` | string | карточка | Короткий район для карточки. |
| `address` | string | страница | Полный адрес. |
| `since` | string | оба | Год основания («2023»). Может быть пустым. |
| `photo` | string | оба | Путь к обложке `assets/images/<id>/cover.jpg`. |
| `gallery` | string[] | страница | Пути к фото галереи. Пусто — блок не показывается. |
| `shortDesc` | string | карточка | Краткое описание (2–3 строки) для карточки. |
| `description` | string | страница | Полное описание производства. |
| `founderStory` | string | страница | История/путь основателя. Пусто — блок скрыт. |
| `products` | string[] | оба | Список продукции (теги). В карточке — первые 4. |
| `productsDetail` | string | страница | Развёрнутое описание продукции. |
| `highlights` | string[] | страница | Преимущества (галочки). Пусто — блок скрыт. |
| `phone` | string | оба | Основной телефон. |
| `phones` | string[] | страница | Все телефоны. |
| `email` | string | страница | Почта. |
| `website` | string | оба | Сайт (полный URL). |
| `social` | object | страница | `{ vk, telegram, other[] }`. |
| `map` | string | страница | Ссылка на карту (Яндекс/Гугл). |
| `photosSource` | string | служ. | Где лежат исходники фото (Я.Диск и т.п.). |
| `status` | object | служ. | `{ called: bool, ready: bool }` — статус отзвона / готовности карточки. |
| `notes` | string | служ. | Что уточнить у производителя. |

## Минимальный валидный объект
```json
{
  "id": "novaya-ferma",
  "icon": "🥕",
  "name": "Название",
  "owner": "ФИО",
  "category": "greens",
  "categoryLabel": "Зелень и салаты",
  "district": "Район",
  "address": "Полный адрес",
  "since": "2024",
  "photo": "assets/images/novaya-ferma/cover.jpg",
  "gallery": [],
  "shortDesc": "Короткое описание для карточки.",
  "description": "Полное описание.",
  "founderStory": "",
  "products": ["Продукт 1", "Продукт 2"],
  "productsDetail": "Описание продукции.",
  "highlights": [],
  "phone": "+7 900 000-00-00",
  "phones": ["+7 900 000-00-00"],
  "email": "",
  "website": "",
  "social": { "vk": "", "telegram": "", "other": [] },
  "map": "",
  "photosSource": "",
  "status": { "called": false, "ready": false },
  "notes": ""
}
```

## Правила
- `id` — уникальный и неизменный (на него завязаны URL и фото).
- `category` ∈ `categories[].slug`.
- Пустые опциональные блоки (`founderStory`, `highlights`, `gallery`, контакты) автоматически скрываются на странице.
- Любой текст экранируется при выводе — HTML внутри значений не интерпретируется.
