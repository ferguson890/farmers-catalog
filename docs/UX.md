# Рекомендации по UI/UX

## Принципы
- **Минимализм и воздух.** Много свободного пространства, мало линий и теней. Фон кремовый/белый, акцент — один зелёный. Никакого визуального шума.
- **Акцент на фото.** Фотография производителя — главный элемент карточки и страницы. Поэтому требования к качеству фото высокие (см. `PERFORMANCE.md`).
- **Качественная типографика.** Контраст serif-заголовков (Playfair Display) и spокойного sans-текста (Lato). Чёткая иерархия размеров.
- **Предсказуемость.** Одинаковые карточки, единый паттерн «фото → название → район → описание → теги → телефон/подробнее».

## Каталог
- **Фильтр по категориям** — горизонтальные вкладки сверху, активная подчёркнута зелёным. Категория «Все» — по умолчанию.
- **Живой поиск** — мгновенно фильтрует по названию, району, продукту, производителю. Без кнопки «искать».
- **Счётчик результатов** — «Показано: N» даёт обратную связь по фильтрам.
- **Пустое состояние** — дружелюбное «Ничего не найдено» с подсказкой сменить запрос.
- **Состояние загрузки** — плейсхолдер, чтобы не было «прыжка» пустого экрана.
- **Hover-микроанимации** — лёгкий подъём карточки, зум фото, сдвиг стрелки «Подробнее». Тонко, не отвлекает.

## Карточка
- Вся карточка кликабельна (ведёт на страницу), но телефон — отдельная ссылка `tel:` (клик не открывает страницу). Это убирает конфликт интеракций.
- Бейдж категории и год основания — поверх фото, не занимают текстовое место.
- Описание ограничено 3 строками (`line-clamp`) — карточки одинаковой высоты, сетка ровная.

## Страница производителя
- **Крупная обложка** + карточка-заголовок, наезжающая на фото (приём из премиум-лендингов).
- **Логичные секции:** О производстве → Основатель → Продукция → Преимущества → Галерея → Контакты. Пустые секции скрываются — нет «дыр».
- **Контакты собраны в один блок:** крупный телефон (главное действие), сайт, почта, соцсети, карта.
- **Возврат в каталог** — заметная ссылка сверху и снизу.

## Доступность (a11y)
- Контраст текста к фону соответствует WCAG AA (тёмный текст на светлом).
- У всех изображений есть `alt`.
- Ссылки — настоящие `<a>` (работают клавиатура, новая вкладка, индексация).
- Тач-цели ≥ 44px на мобильном.
- Внешние ссылки — `rel="noopener"`.

## Чего избегать
- Каруселей-автопрокруток, всплывающих поверх контента баннеров.
- Более одного-двух акцентных цветов.
- Тяжёлых теней и градиентов (кроме лёгкого декоративного пятна в герое).
- Длинных нечитаемых «простыней» — описания структурируем.
